## 5. 联调顺序建议
1. 执行 SQL 初始化脚本并确认初始角色/权限已加载。
2. 先走认证链路（注册、忘记密码、登录）。
3. 再走后台账号角色分发流程与越权校验。
4. 最后验证公告/活动/报修与日志查询。
