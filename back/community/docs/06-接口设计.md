## 3. 认证接口
| 接口 | 方法 | 权限 | 说明 |
|---|---|---|---|
| `/api/auth/login` | POST | 无 | 登录获取 Token |
| `/api/auth/logout` | POST | 已登录 | 登出并拉黑 Token |
| `/api/auth/me` | GET | 已登录 | 获取当前用户信息 |
| `/api/auth/password` | PUT | 已登录 | 修改密码 |

## 9. 日志查询接口（阶段5）

### 9.1 登录日志分页查询
- 接口：`GET /api/logs/logins`
- 权限：`log:login:list`
- 查询参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `current` | long | 否 | 当前页，默认 1 |
| `size` | long | 否 | 每页条数，默认 10，最大 100 |
| `username` | string | 否 | 用户名模糊查询 |
| `successFlag` | int | 否 | 登录结果：1成功/0失败 |
| `ip` | string | 否 | IP 模糊查询 |
| `startTime` | datetime | 否 | 开始时间，格式 `yyyy-MM-dd HH:mm:ss` |
| `endTime` | datetime | 否 | 结束时间，格式 `yyyy-MM-dd HH:mm:ss` |

### 9.2 操作日志分页查询
- 接口：`GET /api/logs/operations`
- 权限：`log:operation:list`
- 查询参数：

| 参数 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `current` | long | 否 | 当前页，默认 1 |
| `size` | long | 否 | 每页条数，默认 10，最大 100 |
| `username` | string | 否 | 用户名模糊查询 |
| `operationModule` | string | 否 | 操作模块精确匹配 |
| `operationType` | string | 否 | 操作类型精确匹配 |
| `successFlag` | int | 否 | 执行结果：1成功/0失败 |
| `traceId` | string | 否 | 链路追踪ID |
| `startTime` | datetime | 否 | 开始时间，格式 `yyyy-MM-dd HH:mm:ss` |
| `endTime` | datetime | 否 | 结束时间，格式 `yyyy-MM-dd HH:mm:ss` |

## 10. 核心状态值
- 公告：`DRAFT/PUBLISHED/RECALLED`
- 活动：`DRAFT/PUBLISHED/RECALLED/FINISHED`
- 报修：`PENDING_ACCEPT/ACCEPTED/ASSIGNED/PROCESSING/WAIT_CONFIRM/DONE/CLOSED/REJECTED/REOPENED`

## 11. 通用错误码（示例）
- `UNAUTHORIZED`：未登录或 Token 失效。
- `FORBIDDEN`：无权限访问。
- `VALIDATE_FAILED`：参数校验失败。
- `ILLEGAL_STATE_FLOW`：非法状态流转。
- `BIZ_ERROR`：业务异常。
